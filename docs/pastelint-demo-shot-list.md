# PasteLint Demo Shot List

## Purpose

This is a practical list of demo shots, screenshots, and short videos to capture from PasteLint using safe synthetic examples from `docs/pastelint-demo-copy-bank.md`.

Use it to plan lightweight product proof: show the messy input, run the tool, and capture the cleaned or prepared result without using private, customer, patron, or production text.

## Recording Rules

- Use synthetic text only.
- Start with the messy input visible.
- Show the cleanup action.
- Show the output/proof state.
- Keep clips short: 15-45 seconds.
- Avoid claiming PasteLint rewrites like an AI tool; it prepares text.
- Verify output manually before quoting exact before/after wording.

## Core Demo Shots

### 1. Clean ChatGPT Output

**Tool/page:** `clean-chatgpt-output.html`

**Demo input source from the copy bank:** AI Meeting Follow-Up Slop

**What to show on screen:**

- Paste the messy follow-up text into the input.
- Click `Clean text`.
- Show the cleaned output and the proof/status cue.
- If useful, show the `Rewrite in SecondDraft` next step without clicking it.

**What to say or caption:** "Before you send AI text, clean the paste."

**Success cue to look for:** Output appears, status confirms the text was cleaned, and the SecondDraft handoff is visible.

**Notes:** Focus on overused phrasing, rough spacing, and cleanup-before-rewrite. Verify exact output manually before quoting it.
