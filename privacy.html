Privacy Policy

Last updated: 2026-04-27

PasteLint is designed as a privacy-first, browser-based text utility site. The core tools are intended to run in your browser without requiring a login, file upload, backend account, or AI API call.

Text you enter

Text entered into PasteLint tools is processed locally in your browser by client-side JavaScript. The site is designed so your pasted text is not uploaded to a PasteLint server.

No accounts

PasteLint does not require user accounts, usernames, passwords, or login credentials.

No intentional text storage

PasteLint does not intentionally store the text you paste into its tools. You should still avoid pasting sensitive information into any web tool unless you are comfortable doing so.

Hosting and third parties

This site may be hosted on GitHub Pages or another static hosting provider. Hosting providers may receive basic technical information such as IP address, browser type, device information, referring pages, and request logs as part of normal web delivery.

Analytics

PasteLint uses basic, privacy-conscious analytics to understand page visits and improve the site. Analytics are for page-level usage only. PasteLint does not send the text you paste into the tools to an analytics service.

Important reminder

PasteLint tools can help clean, rewrite, and format text, but you should review results before using them for factual, legal, medical, financial, professional, or sensitive communication.
